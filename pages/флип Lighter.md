- Вот **супер-сухая выжимка** всей темы:
  
  ```
  Стратегия:
  - маржа 10$
  - плечо 50–200x
  - стоп = -10% маржи (-1$)
  - тейк = +30–50% маржи (+3–5$ и выше)
  - цель: забрать микро-движение, закрыть быстро
  - комиссия = 0 → математическое преимущество
  ```
  
  ---
- ## Математика
  
  ```
  EV_% = p * t - (1 - p) * s
  p_BE = s / (s + t)
  
  При s=10%:
  t=30% → p_BE = 25%
  t=40% → p_BE = 20%
  t=50% → p_BE ≈ 16.7%
  
  Рабочее окно:
  - средний тейк t = 30–50%
  - winrate p = 35–40%+
  → депозит растёт
  ```
  
  ---
- ## Плечо
  
  ```
  Плечо не меняет формулу EV
  Меняет:
  - насколько малое движение цены нужно для прибыли
  - волатильность результата
  - частоту выбивания стопом
  ```
  
  ---
- ## Входы
  
  **Оптимально входить только когда есть микро-перекос:**
  
  ```
  1) Импульс уже идёт
  2) Откат неглубокий, структура не пробита
  3) Объём растёт в сторону входа
  4) Стакан дисбалансирован (bid или ask сильнее)
  5) Спред нормальный
  ```
  
  Нет перекоса → нет входа.
  
  ---
- ## Где брать сигнал
  
  ```
  - orderbook (bid/ask дисбаланс)
  - агрессивный объём по ленте
  - синхронность нескольких бирж
  - импульс + подтверждение отката
  ```
  
  Не свечи → **поток заявок**.
  
  ---
- ## Идея с 2–3 биржами
  
  ```
  Снимаем поток с Binance/Bybit/OKX →
  ищем микро-дисбаланс →
  входим на Lighter без комиссии
  ```
  
  Теоретически рабочая модель.
  
  Это **signal arbitrage / latency alpha**.
  
  ---
- ## Главные скрытые подвохи
  
  ```
  1) задержка данных → ты смотришь в прошлое
  2) проскальзывание → скрытая комиссия
  3) спред шире, чем кажется
  4) стакан Lighter тоньше → хуже исполнение
  5) микро-шипы выбивают стопы
  6) биржа синхронизирует цену и перекос исчезает
  7) конкуренция с HFT/маркет-мейкерами
  8) сильная зависимость от времени и волатильности
  ```
  
  Не теория убивает, а **микромеханика исполнения**.
  
  ---
- ## Что повышает winrate
  
  ```
  - фильтры направления (EMA/структура)
  - фильтры объёма (импульс подтверждён)
  - фильтры ликвидности (топ пары, плотный стакан)
  - вход на откате, не на свечке
  - частичные тейки → увеличивает средний t
  - торговля только в "живые" часы
  - пропуск большинства сигналов
  ```
